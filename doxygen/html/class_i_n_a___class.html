◆ begin()

uint8_t INA_Class::begin ( const uint8_t  maxBusAmps,
const uint32_t  microOhmR,
const uint8_t  devNo = UINT8_MAX 
)

Class initializer.

Instantiates the class. Searches the I2C bus for all INAxxx devices and set the same maxBusAmps and microOhmR values for all of them. If the optional third parameter, devNo, is specified that specific device gets the two specified values set for it. Can be called multiple times, but the 3 parameter version will only function after the 2 parameter version finds all devices

Parameters
[in]maxBusAmpsInteger value holding the maximum expected bus amperage, this value is used to compute a device's internal power register
[in]microOhmRShunt resistance in micro-ohms, this value is used to compute a device's internal power register
[in]devNoDevice number to explicitly set the maxBusAmps and microOhmR values, by default all devices found get set to the same initial values for these 2 parameters
Returns
The integer number of INAxxxx devices found on the I2C bus

Adjust the I2C bus speed.

I2C allows various bus speeds, see the enumerated type I2C_MODES for the standard speeds. The valid speeds are 100KHz, 400KHz, 1MHz and 3.4MHz. Default to 100KHz when not specified.
